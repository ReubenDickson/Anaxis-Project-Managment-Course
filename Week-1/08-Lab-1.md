Software Project Failure Analysis
Objective: To understand real reasons software projects fail and begin thinking like a project manager.

Instructions

Think of any real or hypothetical software project (school system, banking app, e-commerce site, etc.).

Answer the following questions in writing:

i. What was the goal of the project?

ii. What went wrong?

iii. Which failures were management-related?

iv. How could better project management have improved the outcome?


Deliverable

Create a file named:

docs/project-failure-analysis.md


Write at least 300 words.
