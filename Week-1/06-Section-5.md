Introduction to the Capstone Project (CBT Application)


Throughout this course, you will manage a Computer-Based Testing (CBT) application.

Important facts:

- The backend already exists

- There is no frontend

- There is no backlog (to-do list, lists of tasks to be carried out)

- There is no sprint plan

- There is no documented roadmap


This mirrors real-world jobs where:

- Systems already exist

- Documentation is incomplete

- Expectations are high


Your responsibility is to:

- Treat this as a fresh project

- Plan frontend delivery

- Manage work using Agile and GitHub

- Deliver a frontend MVP by the end of the course
