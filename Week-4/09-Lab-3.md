LAB 3 — Define System Requirements

Objective: To formally document what the CBT frontend must deliver.

Instructions

Create a file:
``` bash
docs/requirements.md
```

Your document must include:

1. Functional Requirements
List at least 15 functional requirements
Number each requirement (FR-1, FR-2, …)

2. Non-Functional Requirements
List at least 8 non-functional requirements
Number each requirement (NFR-1, NFR-2, …)

Write clearly and professionally.
