User Journeys

A user journey describes the steps a user takes to achieve a goal. It is a visual, step-by-step mapping of the interactions, emotions, and actions a user experiences when engaging with a product or service, from initial awareness to final goal achievement. It enables Project Managers to identify pain points, optimize touchpoints and improve user satisfaction.

Example (Student):

i. Logs in

ii. Views available exams

iii. Starts an exam

iv. Submits answers

v. Views results

User journeys help you:

i. Spot missing requirements

ii. Identify edge cases

iii. Understand system flow
