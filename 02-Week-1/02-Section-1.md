What Is a Project?

A project is a temporary effort undertaken to create a unique product, service, or result.

Key characteristics of a project:

- Has a clear start and end

- Has defined objectives

- Has constraints (time, scope, cost, quality)

- Produces something new


Examples of Projects

- Building a website

- Developing a mobile application

- Migrating a system to the cloud

- Creating a software testing platform (like your CBT project)


What Is NOT a Project

- Daily server monitoring

- Customer support

- Routine maintenance

Those are operations, not projects.
