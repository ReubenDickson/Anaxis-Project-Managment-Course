Create User Story Issue Template

Objective: To standardize how work is documented.

Instructions

In your repository, create a folder:
``` bash
.github/ISSUE_TEMPLATE
```

Inside it, create a file:
``` bash
user-story.md
```

Include:

i. Story description

ii. Acceptance criteria

iii. Notes

iv. Dependencies (if any)

This template will be reused throughout the project.
